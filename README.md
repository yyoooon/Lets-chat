# Let's chat
실시간 채팅을 할 수 있는 웹앱으로, 다수와의 채팅 또는 1:1 채팅을 할 수 있습니다.

## 기술 스택 

**Back**
- nodeJS
- express
- socket.io

**Front**
- nextjs
- emotion
- recoil
